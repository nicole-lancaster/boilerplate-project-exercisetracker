Create a New User

Add exercises

POST /api/users/:_id/exercises

GET user's exercise log: GET /api/users/:_id/logs[?from=yyyy-mm-dd&to=yyyy-mm-dd&limit=number]

[ ] = optional

If using date queries, you must give both 'from' and 'to' dates. You can pass a limit query with or without a date query.
